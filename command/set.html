Set

set은 변수와input기능을 합친것입니다.
/a는 변수선언이고 /p는 input기능에 입력한값을 변수에 넣는것이다.
그리고 변수는 %과%안에 변수이름을넣으면된다.


명령어↓
----------------------------------------------------------
rem set /a 사용
rem 상황:a라는변수에 1이라는값을넣음
set /a a = 1
echo %a%
rem set /p 사용
rem 상황: 이름을입력하고 그이름을 출력
set /p name = 이름이 무엇입니까?
echo 이름이%name%이군요!
