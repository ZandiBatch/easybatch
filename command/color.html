Color

color는 배경,글자색을바꿉니다.
첫번쨰는 배경,두번째는 글자색입니다. 색은16진수로 바꿉니다.
색코드↓
0 = 검은색 8 = 회색
1 = 파란색 9 = 연한 파란색
2 = 녹색 A = 연한 녹색
3 = 청록색 B = 연한 청록색
4 = 빨간색 C = 연한 빨간색
5 = 자주색 D = 연한 자주색
6 = 노란색 E = 연한 노란색
7 = 흰색 F = 밝은 흰색


명령어↓
----------------------------------------------------------
rem color사용법
color 07
rem 배경색은검정,글자색은 흰색으로설정합니다.
